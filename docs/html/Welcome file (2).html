Shiden Networkとは

Shiden Networkは、Astar Network の研究開発チェーンとして機能する Kusama 上のスマートコントラクトプラットフォームプロジェクトです。

Kusamaとは

KusamaはPolkadotのR&Dチェーンです。つまりPolkadotの研究開発を目的とした独自のブロックチェーンです。

Shiden Networkは、このKusamaに接続するパラチェーンです。

Shiden Networkの特徴

Shiden Networkは、イーサリアム仮想マシン(EVM)、WebAssembly、および DeFi、NFTなどのさまざまなアプリケーションを可能にする複数のレイヤー2ソリューションを最初からネイティブにサポートします。
